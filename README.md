# Realtime Face Avatar Prototype

Protótipo de sistema de avatar 2D controlado por rastreamento facial em tempo real.

O projeto usa:

- MediaPipe (Face Landmarker)
- OpenCV
- Pygame

O objetivo é detectar a rotação da cabeça e trocar automaticamente o sprite do personagem (frente, perfil esquerdo e perfil direito).

---

## Requisitos

- Python 3.11+
- Webcam
